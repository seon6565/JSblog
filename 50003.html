4. Java String

Java String

String 변수명 = new String("문자열") new String 생략가능

특징 1 객체 안의 값을 변경하면 새로운 객체를 생성

특징 2 리터럴을 바로 입력한 경우 문자열이 같을 때 하나의 객체를 공유

length() 문자열의 길이

charAt() 인덱스 위치에서의 문자

indexOf() 첫번째 문자열의 위치 (없을 경우 -1)

lastIndexOf() 뒤에서 첫번째 문자열의 위치 (없을 경우 -1)

.valueOf() 문자열로 변환하기 위한 메서드

concat() 문자열 연결

getBytes() 문자열을 byte[]로 변환

toCharArray() 문자열을 char[]로 변환

toLowerCase() 영문 문자를 모두 소문자로 변환

toUpperCase() 영문 문자를 모두 대문자로 변환

replace() 문자열을 , 문자열로 대체한 문자열 생성

substring() beginIndex부터 endIndex-1 위치까지의 문자열 생성

split() 조건에 맞춰서 문자열을 분할한 배열을 생성

trim() 문자열 앞 뒤 공백 제거

equals() 문자열의 실제 내용 비교

equalsIgnoreCase() 대소문자 구분 없이 실제 내용 비교
