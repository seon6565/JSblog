3. 프론트엔드, 백엔드

프론트엔드와 백엔드

프론트엔드(Front-end)는 사용자 앞에 보이는 영역으로, 웹 브라우저 화면에
보이는 것을 다룹니다. 웹사이트를 디자인 하거나 사용자 동작에 반응하는 기능을 만들며,
프론트 개발에는 HTML, CSS, JavaScript를 사용합니다.

백엔드(Back-end)는 사용자 뒤에서 보이는 영역으로, 웹 사이트의 개인 정보, 게시판 글 등은
서버에 데이터베이스 형태로 저장됩니다. 이러한 데이터 베이스를 설계하거나 데이터를 처리하는 것이
백엔드 개발의 영역이며,
백엔드 개발에는 Java, PHP, Python 등의 다양한 프로그래밍 언어를 사용합니다.

*풀스택(Full-stack)은 프론트엔드와 백엔드 개발을 모두 할수 있는 사람을 의미합니다.
