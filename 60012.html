13. 인덱스, 암호화

인덱스(색인), 암호화

클러스터형 색인 Primary key (1개만 허용)

넌클래스터형 색인 1개이상 생성 가능 (유니크 색인, 넌유니크 색인)

Create Index 인덱스명 on 테이블명(칼럼)

Show index from 테이블명

Show table status like '%aaa%' 테이블 row수 등 분석 정보

Analyze Table 테이블명 색인통계정보 업데이트 명령

Explain select * from 테이블명 select문 관련 분석 정보

SHA2('문자열',256) 256비트로 암호화 (복호화 불가)

HEX(AES_ENcrypt(문자열,대칭키문자열)) 16진수로 변환 (깨짐방지) (복호화 가능)

UNHEX(AES_DEcrypt(암호화된문자열,대칭키문자열)) 16진수로 변환된 암호 복호화
