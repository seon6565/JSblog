10. Java inner class, inner interface

Java inner class, inner interface

inner class의 종류 멤버 이너 클래스(인스턴스 멤버 클래스, 정적 멤버 클래스), 지역 이너 클래스(지역 클래스)
*이너 클래스는 아우터클래스$이너클래스.class 파일이 생성된다.

인스턴스 이너 클래스 생성
아우터클래스 아우터클래스참조변수 = new 아우터 클래스();
아우터클래스.이너클래스 이너클래스참조변수 = 아우터클래스참조변수.new 이너클래스();

정적 이너 클래스 생성
아우터클래스.이너클래스 이너클래스참조변수 = new 아우터클래스.이너클래스();

지역 이너 클래스 지역 이너 클래스는 클래스의 멤버가 아닌 메서드 내에서 정의되는 클래스

지역 이너 클래스 생성 지역이너클래스 지역이너클래스참조변수 = new 지역이너클래스();
*지역 변수를 사용할 때는 반드시 final로 선언(컴파일러가 강제로 추가)

지역 이너 클래스 파일명 아우터클래스$숫자지역이너클래스명.class

익명 이너 클래스 class A { C b = new C() { public void bcd(){ 메서드구현} } };
*인터페이스나 추상클래스를 상속받아 구현하는 익명 클래스를 이너 클래스로 만든다고 생각하면 된다.

inner interface 는 static 이너 인터페이스만 존재할수 있다.

inner interface 객체 생성 방법

1. 인터페이스 구현 클래스 생성 및 객체 생성 class C implements A.B { public void bcd(){메서드구현} } C c = new C(); c.bcd();

2. 익명 이너 클래스 사용 A.B a = new A.B(){ public void bcd() {메서드구현} }; a.bcd();
