3. Java 배열

Java 배열

자료형[] 변수명 = new 자료형[배열의 길이];

1. 자료형[] 변수명 = new 자료형[배열의 길이]; 참조 변수명[0] = 값;배열 객체를 생성하고 값 대입

2. 자료형[] 변수명 = new 자료형[]{값,값,....,값};배열 객체 생성과 함께 값 대입

3. 자료형[] 변수명 = {값,값,....,값};값 바로 대입

1. 자료형[][] 변수명 = new 자료형[배열의 길이][배열의 길이]; 참조 변수명[0][0] = 값;배열 객체를 생성하고 값 대입

2. 자료형[][] 변수명 = new 자료형[배열의 길이][]; 참조 변수명[0] = 배열 생성;배열 객체의 행만 생성 후 차 후 열 생성

3. 자료형[][] 변수명 = new 자료형[][]{값,값,....,값}{값,값,....,값}{값,값,....,값};배열 객체 생성과 함께 값 대입

4. 자료형[][] 변수명 = {값,값,....,값},{값,값,....,값},{값,값,....,값};값 바로 대입
