9. input 태그 속성

input 태그 속성

<autofocus>는 웹 문서가 열리면 해당 필드에 포커스가 됩니다.

<placeholder>는 텍스트 필드에 힌트를 표시합니다.

<readonly>는 입력이 불가능 한 읽기 전용 필드가 됩니다.

<required>는 필수 입력 필드로 지정합니다.

<max>숫자 입력 필드에서 최대값을 지정합니다.

<min>숫자 입력 필드에서 최소값을 지정합니다.

<step>숫자 입력 필드에서 증감할 간격을 지정합니다.

<maxlength>텍스트 관련 필드에서 입력할 수 있는 최대 길이를 지정합니다.

<minlength>텍스트 관련 필드에서 입력할 수 있는 최소 길이를 지정합니다.

<size>텍스트 관련 필드에서 화면에 표시할 크기를 지정합니다.
