5. out, application, exception 객체

out, application, exception 객체

out 객체의 주요 메서드

out.print("버퍼에저장")

out.clearBuffer() 버퍼 비우기

out.getBufferSize() 현재 버퍼에 저장된 사이즈

out.getRemainning() 전체버퍼사이즈-현재버퍼에저장된사이즈

out.flush() 버퍼에 저장된 내용 출력(별다른 설정이 없을 시 버퍼가 가득차면 자동flush)

application 객체의 주요 메서드

application.getRealPath("/프로젝트명") 물리적 경로 방법1

this.getServletContext().getRealPath("/프로젝트명") 물리적 경로 방법2

app.getRealPath("/프로젝트명") 물리적 경로 방법3

application.getInitParameter("INIT_PARAM") 초기화 변수

exception 객체의 주요 사용법

web.xml에 에러페이지,코드,로케이션 태그 추가
<error-page>
<error-code> 404(오류코드) </error-code>
<location> /프로젝트명/이동할.jsp</location>
</error-page>

response.getStatus()에러코드 확인하는 메서드
