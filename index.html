HTML에 대해서 알아보자.

HTML(HyperText Markup Language)는 웹 문서를 만드는 기본 언어입니다.
HTML는 웹 브라우저 창에 웹 문서의 내용을 보여주는데 필요한 약속이라고 할수 있으며
제목,본문,이미지,표와 같은 웹 요소를 알려주는 역할을 합니다.

CSS에 대해서 알아보자.

CSS(Cascading Style Sheets)는 HTML로 만든 내용을 사용자가 알아보기 쉽게 꾸미거나
사용하기 편리하도록 배치할때 사용합니다. 규격이 정해진 PC나 모바일 화면 등에서는
CSS를 이용해서 웹 요소를 적절하게 배치하고 필요한 곳을 강조할 수 있습니다.

JavaScript에 대해서 알아보자.

JavaScript는 사용자의 동작에 반응해서 동적인 효과를 주기 위해서 사용합니다.
JavaScript를 추가하면 웹 문서의 각 요소를 가져와서 필요에 따라 스타일을 변경하거나 움직이게 할 수 있습니다.

웹페이지에 대해서 알아보자.

방문자에게 정보만 보여주는 사이트를 정적사이트(Static site)라고 합니다.
단순히 정보만 보여주는 것이 아니라 사용자의 행동에 반응하여
웹사이트가 동적으로 움직이는 사이트를 동적 사이트(dynamic site)라고 합니다.
