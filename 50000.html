Java에 대해서 알아보자

자바 개발 도구(Java Development Kit)는 자바를 이용해 프로그램을 개발하는데 필요한 도구를 모아둔 것입니다.

자바 실행 환경(Java Runtime Environment)는 완성된 프로그램을 실행하는 데 필요한 환경을 말합니다.

자바 가상 머신(Java Vitual Machine)는 자바를 실행시키기 위한 가상 머신입니다.

*JDK의 설치 위치를 변경했을 경우 환경 변수를 별도로 설정해 줘야 합니다.

.java 소스파일 생성 -> 컴파일 -> .class 바이트 코드 파일 생성 -> 실행 -> .JVM은 메모리 할당 -> 메서드 영역에 클래스 로딩 -> main() 실행
