5. Java 클래스와 객체

Java 클래스와 객체

구조체(struct) 서로 다른 자료형도 1개의 자료형으로 묶어 관리 struct Score {int[] a = {10,20,30}; double b = 78.11 }

클래스(Class) 다양한 자료형의 데이터와 메서드들을 함께 관리

절차지향형 프로그래밍(Procedural Programming) 순서에 맞춰 단계적으로 실행되도록 명령어를 나열

객체지향형 프로그래밍(Object-Oriented Programming) 프로그램이 객체 단위로 수행

클래스(Class) 내부 구성 클래스의 멤버(필드(field), 메서드(method), 이너클래스(inner class))+생성자(constructor)

필드(field) 클래스에 포함된 변수

메서드(method) 클래스가 지니고 있는 기능(함수)

이너클래스(inner class) 클래스 내부의 클래스

생성자(constructor) 클래스의 객체를 생성하는 역할

인스턴스(instance) 클래스의 생성자로 만들어진 객체

클래스명 변수명 = new 생성자();

지역 변수(local variable) 메서드에 포함된 변수

메서드 시그니처(method signature) int sum (int a, int b){ return 3; } 해당 예시에서 sum, int 부분을 메서드 시그니처라 한다.

가변 길이 배열 입력 매개변수 메서드명(자료형...변수명){ } ...을 붙여주면 가변길이로 가능하다.

생성자(constructor) 는 1. 반드시 클래스명과 동일 2. 리턴 타입이 없다.

기본 생성자 는 없으면 컴파일러가 자동으로 추가 함, 생성자는 여러개 가능함

this 자신이 포함된 클래스의 객체를 가리킴(모든 메서드에는 this라는 참조 변수가 있으며, 컴파일러가 자동 추가해준것)

this() 자신이 속한 클래스내부의 다른 생성자를 호출(1. 생성자의 내부에서만 사용 2. 생성자의 첫 줄에 위치)
