14. Java 람다식

Java 람다식

Lambda expression 은 함수형 프로그래밍 기법을 지원하는 자바의 문법 요소

function 는 함수로 기능 또는 동작을 정의한 것이다.

method 는 클래스 또는 인터페이스 내에 정의된 함수이다.

함수형 인터페이스 단 하나의 추상 메서드만을 포함하는 인터페이스

함수형 인터페이스를 상속받아 구현

CASE 1 인터페이스를 상속받은 클래스에서 메서드를 구현하여 실행

CASE 2 익명 함수로 구현 A a = new A(){ void abc(){ 구현 }};

CASE 3 람다식 익명 화살표 함수로 구현 A a = () -> { 구현 };

정의돼 있는 인스턴스 메서드 참조 클래스 객체::인스턴스메서드명
B b = new B(); A a = b::bcd; B객체를 생성 한 후 A인터페이스 내부의 미완성abc()메서드는 b.bcd()실행

정의돼 있는 메서드 참조 클래스명::정적메서드명
A a = B::bcd; B클래스의 static 메서드 bcd()실행

첫 번째 매개변수로 전달된 객체의 메서드를 참조
A a = B::bcd; a.abc(new B(), 3); (bcd는 void bcd(int K){ k출력 })

자바가 제공하는 메서드 참조
A a = String::length; A의 추상메서드 abc는 length()와 동일한 기능을 함.
B a2 = String::length; a2.abc("안녕") B의 추상메서드 abc는 매개변수를 받게 구성되어있음.

배열의 new 생성자 참조 A a = int[]::new;

클래스 생성자 참조 A a = B::new;
