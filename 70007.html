8. JDBC 연동, 커넥션 풀

JDBC 연동, 커넥션 풀

Connection con 커넥션 객체

Class.forName("org.mariadb.jdbc.Driver") 드라이버 로딩

con = DriverManager.getConnection("jidbc:mariadb://loacalhost:3306/maria","ID","PWD") 드라이버매니저에 URL,ID,PWD를 통해 Connection 할당

PreparedStatement psmt = con.perpareStatement("Select * from ?") PreparedStatement 객체 생성

psmt.setString(1,"tbl") ?에 값 할당

psmt.execute(); query실행(execute의 경우 리턴값은 boolean)

커넥션 풀 미리 생성된 것을 가져다 쓰는것

JNDI 미리 설정된 이름으로 찾아쓰는 디렉터리서비스

Server.xml <globalNamingResources> 안에 <Resource 속성들 셋팅 />

Context.xml <ResourceLink> 안에 < 속성들 셋팅 />

Context.lookup("이름") lookup하고자 하는 이름을 context.xml의 golbal속성에서 찾은 후 context.xml의 name속성값으로 server.xml의 name과 매칭시켜줌

Context initctx = new InitialContext(); context 초기 객체 생성

Context ctx = (context)initctx.lookup("java:comp/env") context 경로 연결

DataSource source = (DataSource)ctx.lookup("이름")이름을 통해서 Context.xml->Server.xml에 접근하여 설정 값으로 생성 된 데이터소스를 가져옴

Connection con = source.getConnetion();가져온 데이터소스의 커넥션을 커넥션 객체에 할당
