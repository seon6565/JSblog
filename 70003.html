4. request, response 객체

request, response 객체

request 객체의 주요 메서드

request.getMethod() method 방식 정보 (get/post)

request.getRequestURL() 호스트 포함한 주소

request.getRequestURI() 호스트 미포함한 주소

request.getProtocol() 프로토콜 정보 ex)HTTP/1.1

request.getServerName() 서버이름

request.getServerPort() 포트

request.getRemoteAddr() 클라이언트 IP주소

request.getQueryString() get방식과 같이 URL뒤의 ?부터의 문자열

request.getParameter("속성이름") 속성이름에 해당하는 전송된 값

request.getParameterValues("속성이름") 속성이름에 해당하는 값이 여러개일 경우

request.getHeaderNames() 헤더이름을 가져옴(Enumeration객체)

request.getHeader("헤더이름") 헤더이름에 해당하는 값
*주로쓰는 헤더이름 user-agent(웹브라우저의종류), referer(링크로 접속한 이전경로), cookie(쿠키정보)

request.getRequestDispatcher("포워드할경로").forword(request,response) 포워드할 경로 뒤에 ?속성=값 을 붙여서 보낼수도 있음.

response 주요 메서드

response.sendRedirect("보낼주소")

response.addDateHeader("헤더키","date값")

response.addIntHeader("헤더키","Int값"))

response.addHeader("헤더키","문자값")

response.getHeaderNames()

response.getHeader("헤더키")
