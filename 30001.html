(변수,자료형,연산자,조건문,반복문)

JavaScript (변수,자료형,연산자,조건문,반복문)

변수 프로그램이 실행되는 동안 값이 달라질 수도 있는 데이터

숫자형 숫자로만 표시

문자열 ''나 ""로 표시

논리형 true, false 값만 가짐

배열 하나의 변수에 여러 개의 값이 있음

객체 함수와 속성들이 함께 포함 된 자료형

undefined 데이터 유형이 지정되지 않았을 때의 상태를 나타냄

null 데이터 값이 유효하지 않은 상태를 나타냄

산술 연산자 +, -, *, /, %, ++, --

할당 연산자 =, +=, -=, *=, /=, %=

비교 연산자 ==, !=, ===, !==, <, <=, >. >=

논리 연산자 !, &&, ||

조건 연산자 (조건)?true일때 실행 :false일때 실행

if문 if(조건){ 조건이 true일 때 실행 }

if~else문 if(조건){ 조건이 true일 떄 실행 } else{ 조건이 false일 때 실행 }

switch문 switch(조건){ case 값1: 실행 break; case 값2 실행 break; default: 실행}

for문 for(초기값; 조건; 증가식){ 실행 }

while문 while(조건){ 실행 }

do~while문 do { 실행 } while(조건)

break; 반복문이 끝나기 전 조건에 따라 반복문을 종료

continue; 반복 과정을 생략하고 반복문 처음으로 돌아감
