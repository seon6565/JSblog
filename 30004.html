5. JavaScript DOM

JavaScript DOM

요소 노드 모든 HTML 태그를 의미

텍스트 노드 태그의 텍스트 내용

속성 노드 태그의 속성

주석 노드 주석

getElementById(id) id명으로 접근

getElementByClassName(class) class명으로 접근하며, 배열로 저장

getElementByTagName(tag) 태그명으로 접근하며, 배열로 저장

querySelector() 다양한 방식으로 접근가능

querySelectorAll() 다양한 방식으로 접근가능(단 하나가 아니라, 배열로 받아와야 할 경우 사용)

getAttribute(속성) 태그에서 사용한 속성값을 가져옴

setAttribute(속성, 값) 태그의 속성을 특정한 값으로 지정함

addEventListener(이벤트, 함수, 캡쳐여부) 요소에 특정 이벤트에 대한 콜백함수

createElement() 새로운 요소 노드를 만듬

createTextNode() 새로운 텍스트 노드를 만듬

createAttribute() 새로운 속성 노드를 만듬

setAttributeNode() 속성 노드를 요소 노드의 자식으로 연결

appendChild() 요소 노드를 자식으로 연결

insertBefore(insertThis,beforeThis) 요소를 해당 위치 앞에 추가(null일 경우 마지막에추가)

parent.removeChild(child) 부모 노드에서 자식 노드를 삭제
