7. Java 상속과 다형성

Java 상속과 다형성

상속 부모 클래스의 멤버를 내려받아 자식 클래스 내부에 포함시키는 것

상속 UML 기호 상속에서 UML 기호는 부모 클래스쪽으로 향하게 그린다.

다형성(polymorphism) 객체를 여러가지 모양으로 표현할 수 있는 특성

extends 상속할 떄는 extends 키워드를 사용

class 자식클래스 extends 부모클래스 { } *다중 상속 불가 *생성자는 상속되지 않음

변수 instanceof 타입 캐스팅 가능 여부 확인

메서드 오버라이딩(overriding) 부모 클래스에서 상속받은 메서드와 동일한 이름의 메서드를 재정의

1.부모 클래스의 메서드와 시그니처 및 리턴타입이 동일해야함

2.부모 클래스의 메서드보다 접근 지정자의 범위가 같거나 넓어야함

메서드 오버로딩(overloading) 같은 메소드의 매개변수 중복 정의

super 부모의 객체를 가리킴

super() 부모의 생성자를 호출

Object 자바의 모든 클래스는 별도의 지정이 없으면 Object 클래스를 상속 받은 상태임
