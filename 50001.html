Java 변수와 자료형

변수의 생존 기간 생성된 변수는 자신이 선언된 { }가 끝나면 메모리에서 삭제된다.

기본 자료형 boolean(1byte), byte(1byte), short(2byte), int(4byte), long(8byte), float(4byte), double(8byte), char(2byte)

참조 자료형 배열, 클래스, 인터페이스

자바의 메모리 구조 메모리는 목적에 따라 크게 3가지 영역으로 나뉜다. ㅣ 클래스,정적,상수,메서드영역 ㅣ 스택 영역 ㅣ 힙 영역 ㅣ

boolean 부울대수 자료형(true, false)

byte, short, int, long 정수 자료형

float, double 실수 자료형

char 문자 자료형

업캐스팅(up-casting) 크기가 작은 자료형을 큰 자료형으로 대입

다운캐스팅(down-casting)큰 자료형을 작은자료형으로 대입
