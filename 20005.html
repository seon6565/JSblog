6. CSS 선택자

(연결, 속성, 가상)

CSS 선택자(연결, 속성, 가상)

하위 선택자 상위 요소에 포함된 모든 하위 요소에 적용(상위요소 하위요소{})

자식 선택자 부모 요소에 포함된 자식 요소만 적용(부모요소 > 자식요소{})

인접 형제 선택자 요소1 이후 가장 처음 해당 형제 요소에 적용(요소1 + 요소2{})

형제 선택자 요소1과 형제인 모든 요소에 적용(요소1 ~ 요소2{})

속성 선택자 해당 속성이 있는 요소

속성=값 지정한 속성값이 있는 요소

속성~=값 지정한 속성값이 포함된 요소(단어별 띄어쓰기구분)

속성|=값 지정한 속성값이 포함된 요소(하이픈 포함도구분)

속성^=값 지정한 속성값으로 시작하는 요소

속성$=값 지정한 속성값으로 끝나는 요소

속성*=값 지정한 속성값이 포함되어 있는 요소

:link 방문하지 않은 링크에 적용

:visited 방문했던 링크에 적용

:hover 지정한 요소에 마우스 포인터를 올렸을때 적용

:active 지정한 요소를 활성화 했을때 적용

:focus 지정한 요소에 초점이 맞춰졌을 때 적용

:target 앵커(a)대상에 스타일 적용

:enabled 지정한 요소를 사용할 수 있는 상태일 때 적용

:disabled 지정한 요소를 사용할 수 없는 상태일 때 적용

:checked 선택된 요소에 적용

:not 지정한 요소가 아닐 때 선택해서 적용

:only-child 부모 안에 자식요소가 하나뿐일 떄 자식 요소 선택

:first-child 부모 안에 있는 자식 요소 중 첫 번째 자식 요소를 선택

:last-child 부모 안에 있는 자식 요소 중 마지막 자식 요소를 선택

:nth-child(n) 부모 안에 있는 자식 요소 중 n번째 자식 요소를 선택

:nth-last-child(n) 부모 안에 있는 자식 요소 중 끝에서 n번째 자식 요소를 선택

A:only-type-of 부모 안에 A요소가 하나뿐일 때 자식 요소 선택

A:first-of-type 부모 안에 있는 자식 A요소 중 첫 번째 자식 A요소를 선택

A:last-of-type 부모 안에 있는 자식 A요소 중 마지막 자식 A요소를 선택

A:nth-of-type(n) 부모 안에 있는 자식 A요소 중 n번째 A요소를 선택

A:nth-last-of-type(n) 부모 안에 있는 자식 A요소 중 끝에서 n번째 A요소를 선택

::first-line 첫 번째 줄을 선택

::first-letter 줄에서 첫 번째 글자를 선택

::before 특정 요소의 앞에 추가

::after 특정 요소의 뒤에 추가
