15. Java 입출력

Java 입출력

File 클래스 file(파일), directory(폴더)를 관리하는 클래스

File의 생성자File(String 위치), File(URI uri), File(File parent, String child), File(String parent, String child)
*파일 객체를 생성하는 시점에서는 실제 파일의 존재 유무는 확인하지 않음.(FileNotFoundException 발생할 수 있음)

파일 경로 구분자 windows=\\ mac=/ 공통=File.separator

절대 경로 루트에서부터 특정위치까지의 전체 경로

상대 경로 현재 작업 폴더 위치를 기준으로한 경로

File 클래스의 주요 메서드
isDirectory, isFile(), getName(), getParent()
list()=경로내의 폴더,파일이름을 문자열 배열로 리턴, listFiles()=경로내의 폴더,파일이름을 파일 객체 배열로 리턴
mkdir(), mkdirs(), getAbsolutePath()

ASCII(아스키) 는 American Standard Code for Information Interchange의 약자로 모든 나라의 문자를 표현할 수는 없다.

unicode(유니코드) 는 전 세계 문자를 통합한 문자셋이다. (일단 대표적으로 UTF-8만 알아두자.)

byte 단위 입출력 InputStream, OutputStream (추상클래스)

char 단위 입출력 Reader, Writer (추상클래스)

InputStream의 주요 메서드
int available() 남은 바이트 수를 리턴
abstract int read() int의 하위 1byte에 읽은 데이터를 저장해서 리턴(추상 메서드)
int read(byte[] b)읽은 데이터를 0번째 위치부터 저장하며 읽은 바이트 수를 리턴
int read(byte[] b, int off, int len) len개수 만큼 읽은 데이터를 byte[] b의 off 위치부터 저장
void close() InputStream의 자원 반환

OutputStream의 주요 메서드
void flush() 메모리 버퍼에 저장된 OutputStram 내보내기(실제 출력 수행)
abstract void write(int b) int의 하위 1byte를 output 버퍼에 출력(추상 메서드)
void write(byte[] b) 매개변수로 넘겨진 byte[] b의 0번째 위치에서부터 메모리 버퍼에 출력
void write(byte[] b, int off, int len) byte[]의 off위치에서 len개를 읽은 후 출력
void close() OutputStream의 자원 반환

InputStream 의 자식클래스 FileInputStream, FilterInputStream(BufferedInputStream, DataInputStream)

FileInputStream FileInputStream(File) FileInputStream(String)
*.read() == -1 이면 파일의 끝을 의미.

OutputStream 의 자식클래스 FileOutputStream, FilterOutputStream(BufferedOutputStream, DataOutputStream, printStream)

FileOutputStream FileOutputStream(File) FileOutputStream(File, boolean append) FileOutputStream(String) FileOutputStream(String, boolean append)
*append가 true일때 이어쓰기, false일때 덮어쓰기

System.in, System.out 입력을 위한 InputStream 타입의 객체, 출력을 위한 PrintStream 타입의 객체

BufferedInputStream BufferedInputStream(InputStream) BufferedInputStream(InputStream, int size)

BufferedOutputStream BufferedOutputStream(OutputStream) BufferedOutputStream(OutputStream, int size)

Reader FileReader, BufferedReader, InputStreamReader

Writer FileWriter, BufferedWriter, OutputStreamWriter, printWriter

Reader의 주요 메서드

int skip(long n) n개의 char스킵(스킵된 char 개수 리턴)

int read() int의 하위 2byte에 읽은 데이터를 저장해 리턴

int read(char[] cbuf) 읽은 데이터를 char[] cbuf에 저장하고 읽은 char 개수를 리턴

abstract int read(char[] cbuf, int off, int len) len 개수만큼 읽은 데이터를 char[] cbuf의 off 위치부터 저장(추상 메서드)

abstract void close() Reader의 자원 반환

Writer의 주요 메서드

abstract void flush() 메모리 버퍼에 저장된 데이터 내보내기(실제 출력) (추상 메서드)

void write(int c) int의 하위 2byte를 메모리 버퍼에 출력

void write(char[] cbuf) 매개변수로 넘겨진 char[] cbuf 데이터를 메모리 버퍼에 출력

void write(String str) 매개변수로 넘겨진 String 값을 메모리 버퍼에 출력

void write(String str, int off, int len) str의 off위치에서부터 len개수를 읽어 메모리 버퍼에 출력

abstract void write(char[] cbuf, int off, int len) char[]의 off위치에서부터 len 개수를 읽어 출력(추상 메서드)

abstract void close() Writer의 자원 반환(추상 메서드)

FileReader FileReader(File) FileReader(String)

FileWriter FileWriter(File) FileWriter(File, boolean append) FileWriter(String) FileWriter(String, boolean append)
*append가 true일때 이어쓰기, false일때 덮어쓰기

BufferedReader BufferedReader(Reader) BufferedReader(Reader, int size)
*readLine() 1줄씩 읽어 문자열로 리턴하는 메서드, == null 이면 파일의 끝을 의미.

BufferedWriter BufferedWriter(Writer) BufferedWriter(Writer, int size)

InputStreamReader byte단위를 char단위로 변경
InputStreamReader(InputStream), InputStreamReader(InputStream, Charset cs), InputStreamReader(InputStream, String charsetName)

OutputStreamWriter byte단위를 char단위로 변경
OutputStreamWriter(OutputStream), OutputStreamWriter(OutputStream, Charset cs), OutputStreamWriter(OutputStream, String charsetName)
