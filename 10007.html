8. input 태그 type

input 태그 type

<text>는 한 줄짜리 텍스트를 입력할 수 있는 텍스트 필드입니다.

<password>는 비밀번호를 입력할 수 있는 필드입니다.

<search>는 검색할 때 입력하는 필드입니다.

<url>는 URL 주소를 입력하는 필드입니다.

<email>는 이메일 주소를 입력하는 필드입니다.

<tel>는 전화번호를 입력할 수 있는 필드입니다.

<checkbox>는 주어진 여러 항목에서 2개 이상 선택할 수 있는 체크박스입니다.

<radio>는 주어진 여러 항목에서 1개만 선택할 수 있는 라디오 버튼입니다.

<number>는 숫자를 조절할 수 있는 스핀 박스를 넣습니다.

<range>는 숫자를 조절할 수 있는 슬라이드 막대를 넣습니다.

<date>는 사용자 지역을 기준으로 날짜(연,월,일)를 넣습니다.

<month>는 사용자 지역을 기준으로 날짜(연,월)를 넣습니다.

<week>는 사용자 지역을 기준으로 날짜(연,주)를 넣습니다.

<time>는 사용자 지역을 기준으로 시간(시,분,초,분할 초)를 넣습니다.

<datetime>는 국제 표준시(UTC)로 설정된 날짜와 시간(연,월,일,시,분,초,분할 초)를 넣습니다.

<datetime-local>는 사용자가 있는 지역을 기준으로 날짜와 시간(연,월,일,시,분,초,분할 초)를 넣습니다.

<submit>는 전송 버튼입니다.

<reset>는 리셋 버튼입니다.

<image>는 <submit> 버튼 대신 사용할 이미지를 넣습니다.

<button>는 일반 버튼입니다.

<file>는 파일을 첨부할 수 있는 버튼을 넣습니다.

<hidden>는 사용자에게는 보이지 않지만 서버로는 값을 넘겨주는 필드입니다.
