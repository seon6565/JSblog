3. 데이터 유형

데이터 유형

문자 데이터 유형

char(문자길이) 고정길이 문자형 (0~255바이트)

varchar(문자길이) 가변길이 문자형 (0~65535바이트)

binary(문자길이) 고정길이 이진 데이터 값(0~255바이트)

varbinary(문자길이) 가변길이 이진 데이터 값(0~255바이트)

tinytext (0~255바이트) 문자열 저장

text (0~65,535바이트) 문자열 저장

mediumtext (0~16,777,215바이트) 문자열 저장

longtext (0~4GB) 문자열 저장

blob 대용량의 사진, 동영상 파일을 저장하는데 사용

날짜 데이터 유형

date (3바이트) (YYYY-MM-DD) (1000-01-01~9999-12-31)

time (3바이트) (HH:MM:SS.ssssss) (-838:59:59.999999~838:59:59.999999)

datetime (8바이트) (YYYY-MM-DD HH:MM:SS.ffffff) (1000-01-01 00:00:00.000000~9999-12-31 23:59:59.999999)

timestamp (4바이트) (YYYY-MM-DD HH:MM:SS.ffffff) (1970-01-01 00:00:01(UTC)~2038-01-19 03:14:07(UTC))

year (1바이트) (YYYY) (1901~2155)

숫자 데이터 유형

tinyint (1바이트) (-128~127)or(0~255)

smallint (2바이트) (-32768~32767)or(0~65535)

mediumint (3바이트)

int (4바이트)

bigint (8바이트)

decimal(M,D), numeric(M,D) (M=1~65) M=전체자릿수, D=소수점아래갯수

double(M,D) (8바이트) M=전체자릿수, D=소수점아래갯수

float(M,D) (4바이트) M=전체자릿수, D=소수점아래갯수
