4. CSS 속성

(박스, 테두리, 여백, 레이아웃, 위치 지정)

CSS 속성(박스, 테두리, 여백, 레이아웃, 위치 지정)

width 박스 모델의 너비를 지정

height 박스 모델의 높이를 지정

box-sizing 박스 모델의 크기를 계산할 때 border를 포함할 경우 border-box, 컨텐츠만 계산할 경우 content-box(기본값)

border-style 테두리 스타일 지정

border-width 테두리 두께 지정

border-color 테두리 색상 지정

border-radius 꼭짓점을 둥글게 만듬

margin 마진을 지정

padding 패딩을 지정

display 요소를 배치하는 방법을 지정(block, inline, inline-block, none, grid, flex 등)

float 웹 요소를 왼쪽이나 오른쪽으로 배치(inline으로)

clear float을 해제하여 요소를 다음(아래)로 배치를 원할 경우 사용

left 기준 위치에서 왼쪽에서 떨어지는 범위지정

right 기준 위치에서 오른쪽에서 떨어지는 범위지정

top 기준 위치에서 위에서 떨어지는 범위지정

bottom 기준 위치에서 아래에서 떨어지는 범위지정

position 웹 요소의 위치를 지정(static, relative, absolute, fixed)
