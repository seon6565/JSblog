8. CSS 레이아웃

(flex, grid)

CSS 레이아웃

플렉스 컨테이너 지정 display: (flex, inline-flex)

flex-direction 주축 방향을 지정(row, row-reverse, column, column-reverse)

flex-wrap 컨테이너 너비를 넘을 경우 줄 바꿈 여부(nowrap, wrap, wrap-reverse)

flex-flow 배치 방향과 줄 바꿈을 한번에 지정

justify-content 주축에서 정렬 방법 지정(flex-start, flex-end, center, space-between, space-around)

align-items 교차축에서 정렬 방법을 지정(flex-start, flex-end, center, baseline, stretch)

align-self 교차축에서 개별로 정렬 방법을 지정(flex-start, flex-end, center, baseline, stretch)

align-content 여러 줄 일때 뭉쳐서 정렬 방법을 지정(flex-start, flex-end, center, space-between, space-around,stretch)

그리드 컨테이너 지정 display: (grid, inline-grid)

grid-template-columns 칼럼을 지정(갯수,크기)

grid-template-rows 줄 높이를 지정

grid-column-gap 칼럼 가격을 지정

grid-row-gap 줄 간격을 지정

grid-column-start 칼럼 시작 라인 번호 지정

grid-column-end 칼럼 끝 라인 번호 지정

grid-column 칼럼 시작 번호와 끝 번호(시작번호/끝번호)

grid-row-start 줄 시작 라인 번호 지정

grid-row-end 줄 끝 라인 번호 지정

grid-row 줄 시작 번호와 끝 번호(시작번호/끝번호)

grid-area 탬플릿 이름 지정

grid-template-areas grid-area를 이용해 템플릿 그리드 지정

minmax() 최소값과 최대값을 지정

repeat() 값은 값을 여러번 반복할때 사용(크기,auto-fill,auto-fit)
