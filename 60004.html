5. 내장 함수

내장 함수

LEFT(str, 갯수) 왼쪽에서 갯수만큼 짜름

RIGHT(str, 갯수) 오른쪽에서 갯수만큼 짜름

MID()=SUBSTR()=SUBSTRING(str, pos, len) pos위치부터 개수만큼 짜름(pos생략가능, 위치는 0이아닌 1부터 시작)

SUBSTRING_INDEX(str,찾고자하는것(ex .),갯수) 찾고자하는것의 갯수의 앞까지 짜름

INSTR(str,'찾고자하는것') 찾고자 하는 것의 첫번째 인덱스(1부터 시작, 없으면 0)

LOCATE(str, 칼럼) 칼럼 내의 str의 시작 위치

LCASE()=LOWER(str) 소문자로 만듬

UCASE()=UPPER(str) 대문자로 만듬

LTRIM(str) 왼쪽 공백 제거

RTRIM(str) 오른쪽 공백 제거

TRIM(str) 양쪽 공백 제거

REPLACE(str,befstr,aftstr) str에서 befstr을 aftstr로 변경

CHAR_LENGTH(str) 문자수 반환

LENGTH(str) byte수 반환(한글은 3byte, 알파벳은 1byte)

CONCAT(str1,str2.....) 문자열 합치기

CONCAT_WS('구분문자',str1,str2.......) 구분문자를 추가하면서 합침

CONVERT('바꿀값',바꿀타입(사이즈)) ex. CONVERT(123,VARCHAR(20))

LPAD(str, 길이, 길이될때까지채울것) 길이가 될때까지 채움(미지정시 공백), 짧으면 LEFT()효과

RPAD(str, 길이, 길이될때까지채울것) 길이가 될때까지 채움(미지정시 공백), 짧으면 RIGHT()효과

INSERT(str,인덱스,갯수,넣을문자열) 인덱스에서 개수만큼 짜른뒤 넣을문자열로 대체

REVERSE(str) 문자열 뒤집기 ex. ABCD -> DCBA

SPACE(갯수) 공백 생성

FORMAT(12345.6789,2) 12345.68 (반올림)

ASCII('A') 해당 ASCII값 65(여러글자일 경우 첫글자만 반환)

CHAR(65,66,67) 해당 ASCII의 CHAR반환 ABC

TO_CHAR(날짜데이터,'YYYY-MM-DD') 날짜데이터를 CHAR로 변환(HH시간,MI분,SS초)

ADDDATE(날짜데이터,30) 30일 후(-일 경우 30일전)

NOW(),CURRENT_TIMESTAMP() 현재의 연월일시분초

CURDATE(),CURRENT_DATE() 현재의 연월일

CURTIME(),CURRENT_TIME() 현재의 시분초

YEAR(날짜데이터),MONTH(날짜데이터),DAY(날짜데이터)

HOUR(날짜데이터),MINUTE(날짜데이터),SECOND(날짜데이터),MICROSECOND(날짜데이터)

DATE(날짜데이터),TIME(날짜데이터) 연월일, 시분초

DATEDIFF(1날짜, 2날짜) 1날짜에서 2날짜를 -함

TIMEDIFF(1시간, 2시간) 1시간에서 2시간을 -함

LAST_DAY(날짜데이터) 해당 월의 마지막 일을 반환

QUARTER(날짜데이터) 해당 분기를 반환

DATE_FORMAT(날짜데이터, '%Y/%m/%d') 날짜데이터를 포맷에 맞게 반환

ABS(값) 절대값

CEIL(값), FLOOR(값), ROUND(값) 올림, 버림, 반올림

MOD(값,나눌값) 나머지 반환

PI() PI값 3.14.........반환

POWER(값,승수) 값에 ^승수를 함

RAND(값) 0~1사이의 난수 반환
